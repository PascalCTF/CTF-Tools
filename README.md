## About the project ⚙️
CTF-Generator is a python based tool made for the [`PascalCTF`](https://github.com/PascalCTF) cybersecurity event to create each challenge's directory to be compliant with the [`CTF-Checker`](https://github.com/PascalCTF/CTF-Checker) tool.
